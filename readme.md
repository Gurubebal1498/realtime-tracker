## migrate


## Add App key 

go to streamlab website http://streamlab.io/
then open application get the key and token add them to config/stream_lab.php

```php
return[
    'app_id'=>'',
    'token'=>''
];
```

## Strat

```
 php artisan serve
```

and enjoy
